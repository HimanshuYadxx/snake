## 🚀 Features

- Smooth snake movement  
- Real-time scoring system  
- Game over detection  
- Simple and responsive UI  
- Keyboard controls  

---

## 🕹️ How to Play

- Use **Arrow Keys** to move:
  - ↑ Up
  - ↓ Down
  - ← Left
  - → Right
- Eat the red food block to grow.
- Avoid running into the walls or your own body.
